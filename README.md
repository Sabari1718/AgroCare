AgroCare is a Flutter-based Agritech application designed to help farmers access crop and weather-related insights through a clean and user-friendly interface.

## Features
- Farmer-friendly dashboard
- Crop information and basic advisory
- Responsive UI (mobile & web)
- Provider-based state management

## Weather Module
- Weather data is fetched using a provider-based state management approach.
- In case of API failure or network issues, the app gracefully handles the error state without crashing.
- For assessment purposes, a fallback UI is shown when weather data is unavailable.

## Tech Stack
- Flutter
- Provider (State Management)
